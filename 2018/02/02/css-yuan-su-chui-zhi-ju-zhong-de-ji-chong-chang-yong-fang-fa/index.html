    子元素相对定位

    父元素设置了高度，而且父元素里面只有一个子元素，可使用子元素相对定位。

    .parentElement {
    height: xxpx;
    }

    .childElement {
    position: relative;
    top: 50%;
    transform: translateY(-50%);
    }
